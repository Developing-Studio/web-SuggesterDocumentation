# Unblock
---
### Description
Débloque un utilisateur
### Argument Requis
`utilisateur` - L'utilisateur que vous souhaitez débloquer 
### Utilisation
```
.unblock <utilisateur>
```
### Alias
`allow`
### Permission Requise
Toutes les personnes ayant la permission **Gérer le serveur**, un rôle admin configuré ou un rôle staff configuré.
