# Traductions 

Si vous souhaitez participer à la traduction du projet, vous devez faire partie du programme d'internationalisation. Si vous n'en faites pas encore partie, n'hésitez pas à rejoindre notre serveur de support pour en apprendre plus! Si vous souhaitez faire le grand saut, envoyez un MP à un membre du Staff disant que vous souhaitez aider à la traduction.

#### Comment ça marche ?

Pour la documentation (https://suggester.js.org/#/), nous utilisons GitHub pour gérer les différentes langues que le site supporte, ce qui signifie qu'avoir un minimum d'expérience est une qualité à avoir. Chaque langue aura sa propre branche et le ou les traducteurs devront traduire les fichiers depuis GitHub. Après l'approbation d'un membre du staff, la nouvelle version du site sera disponible directement sur le site. 

Pour Suggester lui-même, nos développeurs ont designé un bot spécialement fait pour faciliter la traduction. Si vous parlez correctement la langue, pouvez comprendre le fonctionnement du bot et bien suivre ses instructions, vous n'aurez aucun souci !

#### J'ai une question, qui puis-je contacter ?

Si vous avez besoin aide, n'hésitez pas à nous contacter par message privé. Les membres du staff seront heureux de vous aider !
